## 5. Aggregation (`reduce`) ü•£

‡πÄ‡∏Ñ‡∏£‡∏∑‡πà‡∏≠‡∏á‡∏°‡∏∑‡∏≠‡∏ó‡∏µ‡πà‡∏ó‡∏£‡∏á‡∏û‡∏•‡∏±‡∏á‡∏ó‡∏µ‡πà‡∏™‡∏∏‡∏î (‡πÅ‡∏•‡∏∞‡∏á‡∏á‡∏ó‡∏µ‡πà‡∏™‡∏∏‡∏î)! ‡πÉ‡∏ä‡πâ **"‡∏¢‡∏∏‡∏ö‡∏£‡∏ß‡∏°"** ‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡πÉ‡∏ô Array ‡πÉ‡∏´‡πâ‡πÄ‡∏´‡∏•‡∏∑‡∏≠ **‡∏Ñ‡πà‡∏≤‡πÄ‡∏î‡∏µ‡∏¢‡∏ß** (‡πÄ‡∏ä‡πà‡∏ô ‡∏ú‡∏•‡∏£‡∏ß‡∏°, ‡∏Ñ‡πà‡∏≤‡πÄ‡∏â‡∏•‡∏µ‡πà‡∏¢, Object ‡∏Å‡πâ‡∏≠‡∏ô‡πÄ‡∏î‡∏µ‡∏¢‡∏ß)

### Syntax:
`array.reduce((‡∏ï‡∏±‡∏ß‡∏™‡∏∞‡∏™‡∏°, ‡∏Ñ‡πà‡∏≤‡∏õ‡∏±‡∏à‡∏à‡∏∏‡∏ö‡∏±‡∏ô) => { ... }, ‡∏Ñ‡πà‡∏≤‡πÄ‡∏£‡∏¥‡πà‡∏°‡∏ï‡πâ‡∏ô)`

```javascript

// ‡∏´‡∏≤‡∏ú‡∏•‡∏£‡∏ß‡∏° (Sum)
const sum = nums.reduce((acc, curr) => {
    console.log(`Accumulator: ${acc}, Current: ${curr}`);
    return acc + curr;
}, 0); // ‡πÄ‡∏£‡∏¥‡πà‡∏°‡∏ï‡πâ‡∏ô‡∏ó‡∏µ‡πà 0

console.log(sum); // 100
```

### ‡∏Å‡∏≤‡∏£‡∏ó‡∏≥‡∏á‡∏≤‡∏ô‡∏ó‡∏µ‡∏•‡∏∞ step:
1. ‡∏£‡∏≠‡∏ö‡πÅ‡∏£‡∏Å: `acc=0`, `curr=10` ‚Üí return `10`
2. ‡∏£‡∏≠‡∏ö‡∏™‡∏≠‡∏á: `acc=10`, `curr=20` ‚Üí return `30`
3. ‡∏£‡∏≠‡∏ö‡∏™‡∏≤‡∏°: `acc=30`, `curr=30` ‚Üí return `60`
4. ‡∏£‡∏≠‡∏ö‡∏™‡∏µ‡πà: `acc=60`, `curr=40` ‚Üí return `100` (‡∏à‡∏ö)

## 6. Method Chaining ‚õìÔ∏è

**‡πÇ‡∏à‡∏ó‡∏¢‡πå:** ‡∏°‡∏µ‡∏™‡∏¥‡∏ô‡∏Ñ‡πâ‡∏≤‡πÉ‡∏ô‡∏ï‡∏∞‡∏Å‡∏£‡πâ‡∏≤ 1. ‡∏Å‡∏£‡∏≠‡∏á‡πÄ‡∏≠‡∏≤‡πÄ‡∏â‡∏û‡∏≤‡∏∞‡∏Ç‡∏≠‡∏á‡∏Å‡∏¥‡∏ô 2. ‡∏•‡∏î‡∏£‡∏≤‡∏Ñ‡∏≤ 50% 3. ‡∏´‡∏≤‡∏ú‡∏•‡∏£‡∏ß‡∏°‡∏£‡∏≤‡∏Ñ‡∏≤‡∏™‡∏∏‡∏ó‡∏ò‡∏¥

```javascript

    { item: "Apple", price: 100, type: "food" },
    { item: "Shirt", price: 500, type: "clothes" },
    { item: "Cake",  price: 200, type: "food" }

    .filter(p => p.type === "food")       // 1. ‡πÅ‡∏¢‡∏Å‡∏Ç‡∏≠‡∏á‡∏Å‡∏¥‡∏ô: [{Apple,100}, {Cake,200}]
```

> **‡πÄ‡∏ó‡∏Ñ‡∏ô‡∏¥‡∏Ñ:** ‡∏Å‡∏≤‡∏£‡πÄ‡∏Ç‡∏µ‡∏¢‡∏ô‡πÅ‡∏ö‡∏ö Chaining ‡∏ó‡∏≥‡πÉ‡∏´‡πâ‡∏≠‡πà‡∏≤‡∏ô‡πÇ‡∏Ñ‡πâ‡∏î‡πÅ‡∏•‡πâ‡∏ß‡πÄ‡∏Ç‡πâ‡∏≤‡πÉ‡∏à Business Logic ‡πÑ‡∏î‡πâ‡∏ó‡∏±‡∏ô‡∏ó‡∏µ‡πÇ‡∏î‡∏¢‡πÑ‡∏°‡πà‡∏ï‡πâ‡∏≠‡∏á‡∏°‡∏≤‡∏ô‡∏±‡πà‡∏á‡πÑ‡∏•‡πà `for` loop ‡∏Ñ‡∏£‡∏±‡∏ö
