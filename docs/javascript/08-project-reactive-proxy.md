‡πÄ‡∏û‡∏∑‡πà‡∏≠‡πÉ‡∏´‡πâ‡∏á‡πà‡∏≤‡∏¢ ‡πÄ‡∏£‡∏≤‡∏à‡∏∞‡πÉ‡∏ä‡πâ `Map` ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏ú‡∏π‡∏Å `Dep` ‡πÄ‡∏Ç‡πâ‡∏≤‡∏Å‡∏±‡∏ö‡πÅ‡∏ï‡πà‡∏•‡∏∞ Key ‡∏Ç‡∏≠‡∏á Object

```javascript

// targetMap Structure: { targetObject: { key: Dep } }
const targetMap = new WeakMap();

function getDep(target, key) {
    // ‡πÇ‡∏Ñ‡πâ‡∏î‡∏™‡πà‡∏ß‡∏ô‡∏ô‡∏µ‡πâ‡πÅ‡∏Ñ‡πà‡∏´‡∏≤ Dep ‡πÉ‡∏´‡πâ‡πÄ‡∏à‡∏≠ ‡∏ñ‡πâ‡∏≤‡πÑ‡∏°‡πà‡∏°‡∏µ‡∏Å‡πá‡∏™‡∏£‡πâ‡∏≤‡∏á‡πÉ‡∏´‡∏°‡πà

    if (!depsMap) {
        depsMap = new Map();
        targetMap.set(target, depsMap);
    }
    
    let dep = depsMap.get(key);
    if (!dep) {
        dep = new Dep();
        depsMap.set(key, dep);
    }
    return dep;
}

function reactive(raw) {
    return new Proxy(raw, {
        get(target, key) {
            const dep = getDep(target, key);
            dep.depend(); // ‚ú® ‡∏û‡∏£‡∏∞‡πÄ‡∏≠‡∏Å 1: ‡∏à‡∏≥‡∏Ñ‡∏ô‡∏≠‡πà‡∏≤‡∏ô
            return target[key];
        },
        set(target, key, value) {
            target[key] = value; // ‡∏≠‡∏±‡∏û‡πÄ‡∏î‡∏ó‡∏Ñ‡πà‡∏≤‡∏à‡∏£‡∏¥‡∏á‡∏Å‡πà‡∏≠‡∏ô
            const dep = getDep(target, key);
            dep.notify(); // ‚ú® ‡∏û‡∏£‡∏∞‡πÄ‡∏≠‡∏Å 2: ‡πÅ‡∏à‡πâ‡∏á‡∏Ñ‡∏ô‡πÉ‡∏ä‡πâ
            return true;
        }
    });
}
```
