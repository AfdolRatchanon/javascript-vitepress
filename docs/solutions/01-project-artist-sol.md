# Solution: Project 1 (The Console Artist)

## Phase 1: Basic Shapes
นี่คือโค้ดสำหรับวาดรูปสี่เหลี่ยมครับ:

```javascript
// file: artist.js
console.log("-------");
console.log("|     |");
console.log("|     |");
console.log("-------");
```

## Phase 2: The Face
นี่คือโค้ดสำหรับวาดรูปหน้าคน พร้อมคำอธิบายเรื่อง Backslash:

```javascript
console.log("  _____  ");
console.log(" /     \\ "); // ต้องใช้ \\ สองตัว เพื่อแสดง \ หนึ่งตัว
console.log("|  o o  |");
console.log("|   ^   |");
console.log("|  ___  |");
console.log(" \\_____/ "); // ตรงนี้ก็เหมือนกัน
```
